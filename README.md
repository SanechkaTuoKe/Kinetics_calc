# Kinetics_calc
Это - калькулятор для пересчета концентраций (CLX/FUR в MgO)
Программа автоматизирует расчеты концентраций и процентов высвобождения фурациллина из композита MgO для различных временных интервалов. Результаты выводятся в формате, готовом для копирования в Excel/Google Таблицы.
Можно получить:
Расчет концентраций по оптической плотности
Поддержка 7 временных точек: 15, 30, 45, 60, 90, 120, 180 секунд
Поддержка 6 времен высвобождения: 1, 2.5, 5, 7.5, 10, 15, 20 минут
Поддержка 9 концентраций: 20, 15, 10, 5, 2.5, 1, 0.5, 0.1, 0.01 мг/мл
Автоматическое формирование таблиц для Excel
Генерация данных для графиков высвобождения

Для фурациллина MgO используется следующая формула:

C = (A + 0.00608) / 27905.64
Где:
C - концентрация фурациллина (мг/мл)

A - оптическая плотность (безразмерная величина)

0.00608 - поправочный коэффициент

27905.64 - калибровочный коэффициент

2. Расчет количества в 6 мл
C_6мл = C × 0.06
Где:

C_6мл - количество фурациллина в 6 мл раствора (мг)

0.06 - коэффициент пересчета
Накопленная сумма высвобождения
Для каждого временного интервала рассчитывается накопленная сумма:

text
Sum_k = Σ(C_6мл_i) для i от 1 до k
Sum_k - общее количество высвободившегося вещества к моменту времени k

Расчет процента высвобождения
text
Процент = (Sum_180 / Общее_количество) × 100%
Где:

Sum_180 - суммарное количество за 180 секунд

Общее_количество = концентрация × 6 мл

Используемые формулы
Линейная калибровочная зависимость: C = k × A + b

Пересчет на объем: учитывается объем пробы 6 мл

Точность расчетов
Все расчеты выполняются с точностью 16 знаков после запятой
